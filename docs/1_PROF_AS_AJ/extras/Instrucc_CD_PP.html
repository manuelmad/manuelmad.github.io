• INSTRUCCIONES
    • En caso de ingresar valores con parte decimal, debe utilizarse como símbolo el punto (.).
    • Deben llenarse filas de datos consecutivas empezando desde la primera, no dejar filas vacías intermedias. En caso de sobrar filas, las que se dejarán vacías deben estar al final.
    • Ingrese valores de Profundidad y Densidad de Poro. La Densidad de Lodo se calculará automáticamente.
    • Ingrese valores de Profundidad y Densidad de Fractura. La Densidad de Margen de Arremetida se calculará automáticamente.
